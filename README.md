# python-atm
A simple Python-based ATM simulation demonstrating core banking operations like balance check, deposit, withdrawal, and transaction tracking.

ATM Simulation Project

This is a simple Python-based ATM simulation that replicates basic banking operations in a command-line interface. It is designed to strengthen fundamental programming concepts and logical thinking through practical implementation.

Objective

The project aims to simulate real-world ATM functionality while practicing core Python concepts such as functions, loops, conditionals, and user input handling.

Features
Check account balance
Deposit money
Withdraw money with balance validation
View transaction history
Menu-driven user interface
Concepts Used
Functions
Loops
Conditional statements
Lists for transaction tracking
Input handling
Working

The program runs in a continuous loop and presents a menu to the user. Based on the selected option, it performs banking operations and updates the balance accordingly. All transactions are recorded and can be viewed through the history option.
